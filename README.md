# NiveshakAI

An AI-powered stock market screener and analysis platform for Indian retail investors — built as a learning project and future startup foundation.

**GitHub:** https://github.com/VanshXKath/Niveshak

**v1.0** includes all 10 feature engines with a **beautiful HTML/CSS web UI** (Groww / broker-style) served directly by FastAPI.

> This is educational software, not financial advice.

## What We Are Building

| Layer | Technology | Role |
|-------|-----------|------|
| Frontend | HTML + CSS (Jinja2 templates) | Broker-style dashboard — **no JavaScript required from you** |
| Backend | FastAPI | REST API + serves the website |
| Data | yfinance | Free market data from Yahoo Finance |
| Charts | Plotly | Interactive candlestick and line charts |

When you search `RELIANCE`, the backend converts it to `RELIANCE.NS`, fetches data from Yahoo Finance, and returns price, PE ratio, volume, market cap, sector info, and chart history.

## Quick Start (Windows PowerShell)

### Option A: Automated setup

```powershell
cd E:\stock_screener_phase1_with_guides
.\scripts\setup.ps1
```

### Option B: Manual setup

```powershell
cd E:\stock_screener_phase1_with_guides
py -3 -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
copy .env.example .env
```

### Run the app (two terminals)

**Terminal 1 — Backend:**

```powershell
.\.venv\Scripts\Activate.ps1
python -m uvicorn backend.app.main:app --reload
```

**Open the website:**

```powershell
python -m uvicorn backend.app.main:app --reload
```

Then visit **http://127.0.0.1:8000/dashboard**

**You only need HTML, CSS & Python.** Pages use HTML forms; Python fetches data and renders templates. Charts are generated by Plotly in Python (you don't write JS).

> Streamlit UI (`frontend/`) remains as optional legacy fallback.

## API Endpoints (Phase 1)

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/` | Health check |
| GET | `/api/stocks/search?q=reliance` | Search NSE stocks by name or symbol |
| GET | `/api/stocks/{symbol}/summary` | Company profile and key metrics |
| GET | `/api/stocks/{symbol}/history?period=6mo` | Historical OHLCV prices |

## Common Issues

| Problem | Fix |
|---------|-----|
| `uvicorn` not found | Use `python -m uvicorn backend.app.main:app --reload` |
| Streamlit cannot reach backend | Start the backend first on port 8000 |
| No stock data returned | Yahoo Finance may rate-limit; wait a few minutes and retry |
| PowerShell blocks venv activation | `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser` |
